เงื่อนไขการอนุญาตให้ใช้สิทธิสำหรับซอฟต์แวร์ของ MICROSOFT

MICROSOFT WINDOWS PC HEALTH CHECK

หากคุณอยู่ใน (หรือเป็นธุรกิจที่มีสำนักงานใหญ่อยู่ใน) สหรัฐอเมริกา โปรดอ่านเนื้อหาในส่วน “การผูกพันตามคำชี้ขาดของอนุญาโตตุลาการ และการสละสิทธิ์ดำเนินคดีแบบกลุ่ม” ด้านล่างนี้ มีผลกระทบต่อวิธีการแก้ไขข้อพิพาท

เงื่อนไขการอนุญาตเหล่านี้เป็นข้อตกลงระหว่างคุณและ Microsoft Corporation (หรือหนึ่งในบริษัทในเครือ) เงื่อนไขเหล่านี้ใช้กับซอฟต์แวร์ที่มีชื่อตามข้างต้น ตลอดจนบริการหรือการปรับปรุงซอฟต์แวร์ใด ๆ ของ Microsoft (ยกเว้นในกรณีที่บริการหรือการปรับปรุงดังกล่าวมาพร้อมกับเงื่อนไขใหม่หรือเพิ่มเติม ซึ่งเป็นกรณีที่มีการใช้เงื่อนไขที่ต่างออกไปเหล่านั้นในอนาคตและไม่เป็นการเปลี่ยนสิทธิของคุณหรือของ Microsoft ที่เกี่ยวข้องกับซอฟต์แวร์หรือบริการที่มีการปรับปรุงก่อนหน้านี้) หากคุณปฏิบัติตามเงื่อนไขการอนุญาตให้ใช้สิทธิเหล่านี้ คุณจะมีสิทธิด้านล่างนี้ เมื่อใช้งานซอฟต์แวร์นี้ จะหมายความว่าคุณยอมรับในเงื่อนไขเหล่านี้

1.    สิทธิในการติดตั้งและใช้งาน

a)    ทั่วไป คุณสามารถติดตั้งและใช้จำนวนใด ๆ ก็ตามของ ชุดสำเนา ของซอฟต์แวร์.

b)    ส่วนประกอบของบุคคลภายนอก ซอฟแวร์อาจรวมถึงส่วนประกอบของบุคคลภายนอกที่มีคำบอกกล่าวทางกฎหมายแยกต่างหากหรืออยู่ภายใต้ข้อตกลงอื่น ตามที่อาจอธิบายไว้ในไฟล์ ThirdPartyNotices ที่แนบมาพร้อมกับซอฟต์แวร์

2.     ข้อมูล

a)    การเก็บรวบรวมข้อมูล ซอฟต์แวร์นี้อาจเก็บรวบรวมข้อมูลเกี่ยวกับคุณและการใช้ซอฟต์แวร์ของคุณและส่งให้กับ Microsoft Microsoft อาจใช้ข้อมูลนี้ในการให้บริการและปรับปรุงผลิตภัณฑ์และบริการของเรา คุณสามารถบอกปฏิเสธไม่ยอมรับการดำเนินการเหล่านี้ได้ในหลายสถานการณ์ แต่ไม่ใช่ทั้งหมด โดยจะเป็นไปตามที่อธิบายไว้ในเอกสารกำกับผลิตภัณฑ์  คุณสมบัติบางอย่างในซอฟต์แวร์นี้อาจช่วยให้คุณสามารถทำการเก็บรวบรวมข้อมูลจากผู้ใช้แอปพลิเคชั่นของคุณได้ หากคุณใช้คุณสมบัติเหล่านี้ในการเก็บรวบรวมข้อมูลในแอปพลิเคชันของคุณ คุณต้องปฏิบัติตามกฎหมายที่ใช้บังคับ รวมถึงการแจ้งให้ผู้ใช้แอปพลิเคชันของคุณทราบอย่างเหมาะสม คุณสามารถเรียนรู้เพิ่มเติมเกี่ยวกับการเก็บรวบรวมข้อมูลและการใช้งานได้ในเอกสารความช่วยเหลือและในคำแถลงด้านความเป็นส่วนตัวที่ https://aka.ms/privacy การใช้ซอฟต์แวร์ของคุณดำเนินการตามความยินยอมของคุณต่อวิธีปฏิบัติเหล่านี้

b)    การประมวลผลข้อมูลส่วนบุคคล ตามขอบเขตที่ Microsoft เป็นตัวประมวลผลหรือตัวประมวลผลย่อยของข้อมูลส่วนบุคคลที่เกี่ยวข้องกับซอฟต์แวร์ Microsoft มีข้อผูกพันกับลูกค้าทุกคนภายใต้ข้อกำหนดด้านการคุ้มครองข้อมูลส่วนบุคคลทั่วไปของสหภาพยุโรปสำหรับเงื่อนไขการให้บริการออนไลน์ ซึ่งมีผลบังคับใช้ในวันที่ 25 พฤษภาคม 2018 ที่ https://docs.microsoft.com/en-us/legal/gdpr

3.    ขอบข่ายการอนุญาตให้ใช้สิทธิ์ ซอฟต์แวร์นี้ได้รับการอนุญาตให้ใช้สิทธิ ไม่ใช่การอนุญาตให้จำหน่าย Microsoft ขอสงวนสิทธิ์อื่น ๆ ทั้งหมด เว้นแต่กฎหมายที่บังคับใช้กำหนดให้คุณมีสิทธิมากขึ้นถึงแม้จะมีข้อจำกัดนี้อยู่ คุณจะไม่ (และไม่มีสิทธิที่จะ):

a)    เข้าไปยุ่งเกี่ยวกับข้อจำกัดทางด้านเทคนิคใด ๆ ในซอฟต์แวร์ที่อนุญาตให้คุณใช้ในรูปแบบเฉพาะ;

b)    ผู้ใช้ที่ได้รับอนุญาตให้ใช้สิทธิจะไม่สามารถทำวิศวกรรมย้อนกลับ (reverse engineer) แยกส่วน หรือแยกองค์ประกอบซอฟต์แวร์ หรือพยายามได้มาซึ่งรหัสต้นทาง (source code) สำหรับซอฟต์แวร์ เว้นแต่และเท่าที่กำหนดไว้โดยเงื่อนไขการอนุญาตให้ใช้สิทธิของบุคคลภายนอกซึ่งใช้บังคับกับการใช้งานส่วนประกอบโอเพนซอร์ส (Open Source) บางอย่างที่อาจรวมอยู่กับซอฟต์แวร์

c)    ลบ ลด ปิดกั้น หรือแก้ไขข้อความแจ้งเตือนใด ๆ ของ Microsoft หรือของซัพพลายเออร์ที่อยู่ในซอฟต์แวร์นี้;

d)    ใช้ซอฟต์แวร์นี้ด้วยวิธีการใด ๆ ที่ละเมิดกฎหมาย หรือนำไปสร้าง หรือเผยแพร่มัลแวร์; หรือ

e)    แชร์ เผยแพร่ แจกจ่าย หรือให้ยืมซอฟต์แวร์ ใชัซอฟต์แวร์นี้เป็นโซลูชันที่เป็นโฮสต์แบบแยกเดี่ยว เพื่อให้ผู้อื่นใช้งาน หรือโอนซอฟต์แวร์หรือข้อตกลงฉบับนี้ให้กับบุคคลภายนอกใด ๆ

4.    ข้อจำกัดในการส่งออก คุณต้องปฎิบัติตามกฎหมายและข้อบังคับด้านการส่งออกทั้งในและระหว่างประเทศที่ใช้บังคับกับซอฟต์แวร์ ซึ่งรวมถึงข้อจำกัดของที่หมายปลายทาง ผู้ใช้ปลายทาง และการใช้ขั้นปลาย สำหรับข้อมูลเพิ่มเติมเกี่ยวกับข้อจำกัดในการส่งออก โปรดดูที่ https://aka.ms/exporting

5.    บริการสนับสนุน Microsoft ไม่มีข้อผูกพันภายใต้ข้อตกลงฉบับนี้ที่จะให้บริการสนับสนุนสำหรับซอฟต์แวร์นี้ การสนับสนุนใด ๆ ที่ให้นั้นเป็น “อย่างที่เป็น” “มีข้อผิดพลาดทั้งหมด” และไม่มีการรับประกันใด ๆ ทั้งสิ้น

6.    การปรับปรุง ซอฟต์แวร์นี้อาจตรวจสอบสำหรับการปรับปรุงและดาวน์โหลดและติดตั้งการปรับปรุงให้กับคุณตามช่วงเวลา คุณสามารถขอรับการปรับปรุงได้จาก Microsoft หรือแหล่งข้อมูลที่ได้รับอนุญาตเท่านั้น Microsoft อาจจำเป็นต้องปรับปรุงระบบของคุณ เพื่อที่จะส่งการปรับปรุงให้กับคุณได้ คุณยินยอมรับการปรับปรุงแบบอัตโนมัติเหล่านี้โดยไม่มีการแจ้งเตือนเพิ่มเติมใด ๆ การปรับปรุงอาจไม่รวมถึงหรือสนับสนุนคุณสมบัติของซอฟต์แวร์ บริการ หรืออุปกรณ์เสริมทั้งหมดที่มีอยู่ในปัจจุบัน

7.    การผูกพันตามคำชี้ขาดของอนุญาโตตุลาการ และการสละสิทธิ์ดำเนินคดีแบบกลุ่ม เนื้อหาในส่วนนี้จะนำไปใช้บังคับ หากคุณอยู่ใน (หรือหากธุรกิจ ที่ตั้งสถานประกอบการหลักของธุรกิจของคุณอยู่ใน) สหรัฐอเมริกา  หากคุณและ Microsoft มีข้อพิพาทระหว่างกัน คุณและ Microsoft ตกลงที่จะแก้ปัญหาร่วมกันอย่างไม่เป็นทางการเป็นระยะเวลา 60 วัน หากคุณและ Microsoft ไม่สามารถตกลงกันได้เอง คุณและ Microsoft ตกลงที่จะยอมผูกพันตามคำชี้ขาดของอนุญาโตตุลาการ โดยเสนอข้อพิพาทต่อ American Arbitration Association ภายใต้กฎหมาย Federal Arbitration Act (“FAA”) และจะไม่นำไปฟ้องร้องในศาลต่อหน้าผู้พิพากษาและลูกขุน หรือไม่เช่นนั้น องค์อนุญาโตตุลาการกลางจะเป็นผู้ชี้ขาด ไม่อนุญาตให้ดำเนินการกระบวนการพิจารณาคดีแบบกลุ่ม การอนุญาโตตุลาการแบบรวมกลุ่ม การดำเนินคดีโดยคณะเอกชนในนามของสาธารณชน และการดำเนินคดีอื่นใด ๆ ที่บุคคลหนึ่งดำเนินการในนามของตัวแทน; หรือรวมการดำเนินคดีของแต่ละบุคคลเข้าด้วยกันหากไม่ได้รับความยินยอมจากทุกฝ่าย ข้อตกลงการอนุญาโตตุลาการที่สมบูรณ์จะมีเงื่อนไขเพิ่มเติมและมีอยู่ที่ https://aka.ms/arb-agreement-4 คุณและ Microsoft ยินยอมรับเงื่อนไขดังต่อไปนี้

8.    ข้อตกลงทั้งหมด ข้อตกลงฉบับนี้และเงื่อนไขอื่นใดที่ Microsoft อาจกำหนดเพิ่มเติมสำหรับส่วนต่อเติม การปรับปรุง หรือแอปพลิเคชั่นของบุคคลภายนอก คือข้อตกลงทั้งหมดสำหรับซอฟต์แวร์นี้

9.    กฎหมายที่บังคับใช้และสถานที่เพื่อระงับข้อพิพาท หากคุณซื้อหรือได้รับซอฟต์แวร์มาจากในประเทศสหรัฐอเมริกาหรือแคนาดา ให้นำกฎหมายประจำมลรัฐ หรือมณฑลที่คุณอาศัยอยู่ (หรือหากเป็นธุรกิจ ที่สำนักงานหลักของธุรกิจของคุณตั้งอยู่) มาใช้กำกับควบคุมหรือกับการตีความข้อตกลงฉบับนี้ การเรียกร้องให้ชดเชยการผิดสัญญา และการเรียกร้องอื่น ๆ นอกเหนือจากนี้ (รวมถึง การปกป้องผู้บริโภค การแข่งขันอย่างไม่เป็นธรรม และการเรียกร้องจากการถูกละเมิด) โดยไม่คำนึงว่าจะมีความขัดแย้งในข้อกฎหมายหรือไม่ก็ตาม ยกเว้นกรณีที่ FAA ได้เข้ามากำกับดูแลทุกเรื่องที่เกี่ยวกับกระบวนการอนุญาโตตุลาการ หากคุณซื้อหรือได้รับซอฟต์แวร์มาจากประเทศอื่น ให้นำกฎหมายของประเทศนั้นมาใช้ ยกเว้นกรณีที่ FAA ได้เข้ามากำกับดูแลทุกเรื่องที่เกี่ยวกับกระบวนการอนุญาโตตุลาการ หากอยู่ในขอบเขตอำนาจพิจารณาคดีของรัฐบาลกลางสหรัฐฯ คุณและ Microsoft ยินยอมรับเขตอำนาจศาลและสถานที่พิจารณาคดีเฉพาะของศาลรัฐบาลกลาง (federal court) ประจำคิงเคาวน์ตี้ มลรัฐวอชิงตัน (ไม่นับรวมการอนุญาโตตุลาการ) เพื่อไต่สวนข้อพิพาททั้งหมด หากไม่แล้ว คุณและ Microsoft ยอมรับในอำนาจทางกฎหมายที่เป็นสิทธิ์ขาดเฉพาะและสถานที่ในศาลสูงของคิงเคาน์ตี้ มลรัฐวอชิงตัน สำหรับการไต่สวนข้อพิพาททั้งหมดในศาล (ไม่รวมการอนุญาโตตุลาการ)

10. สิทธิของผู้บริโภค; แตกต่างกันไปในแต่ละภูมิภาค ข้อตกลงฉบับนี้อธิบายถึงสิทธิทางกฎหมายบางประการ คุณอาจมีสิทธิอื่น ซึ่งรวมถึงสิทธิของผู้บริโภคภายใต้กฎหมายในรัฐ จังหวัด หรือประเทศของคุณ นอกเหนือจากความสัมพันธ์ของคุณที่มีกับ Microsoft คุณยังอาจมีสิทธิที่พึงได้รับมาจากฝ่ายที่คุณได้ซื้อหรือรับซอฟต์แวร์นี้ ซึ่งเป็นสิทธิที่แยกต่างหาก ข้อตกลงฉบับนี้ไม่ได้เปลี่ยนแปลงสิทธิอื่น ๆ เหล่านั้นของคุณหากกฎหมายในรัฐ จังหวัด หรือประเทศของคุณไม่อนุญาตให้ทำเช่นนั้น ยกตัวอย่างเช่น หากคุณซื้อหรือได้รับซอฟต์แวร์มาจากหนึ่งในภูมิภาคด้านล่างนี้ หรือมีการบังคับใช้กฎหมายของประเทศ ข้อกำหนดดังต่อไปนี้มีผลบังคับใช้กับคุณ:

a)    ออสเตรเลีย คุณได้รับการคุ้มครองตามกฎหมายภายใต้กฎหมาย Australian Consumer Law และไม่มีข้อความใดในข้อตกลงฉบับนี้ที่มีเจตนาที่จะกระทบต่อสิทธิดังกล่าวนั้น

b)    แคนาคา หากคุณซื้อหรือได้รับซอฟต์แวร์นี้ในประเทศแคนาดา คุณสามารถหยุดรับการปรับปรุงได้ด้วยการปิดคุณสมบัติการปรับปรุงโดยอัตโนมัติ หยุดการเชื่อมต่ออินเทอร์เน็ตกับอุปกรณ์ของคุณ (หากและเมื่อคุณเชื่อมต่อกับอินเทอร์เน็ตอีกครั้ง ซอฟต์แวร์จะดำเนินการตรวจสอบเพื่อติดตั้งการปรับปรุงต่อไป) หรือถอนการติดตั้งซอฟต์แวร์ เอกสารที่แนบมากับผลิตภัณฑ์ (ถ้ามี) อาจระบุวิธีปิดการปรับปรุงสำหรับอุปกรณ์หรือซอฟต์แวร์บางชนิดของคุณ

c)    เยอรมนีและออสเตรีย

i.     การรับประกัน ซอฟต์แวร์ที่ได้รับลิขสิทธิ์อย่างถูกต้อง จะทำงานได้ครบถ้วนดังที่ได้ระบุไว้ในเอกสารของ Microsoft ที่ติดมากับซอฟต์แวร์ อย่างไรก็ตาม Microsoft จะไม่ให้การรับประกันตามสัญญาแก่ซอฟต์แวร์ที่ได้รับลิขสิทธิ์ถูกต้อง

ii.    ข้อจำกัดการรับผิด ในกรณีของการประพฤติผิดโดยเจตนา หรือประมาทเลินเล่ออย่างร้ายแรง การเรียกร้องค่าเสียหายตามกฎหมาย Product Liability Act รวมถึงกรณีการเสียชีวิต หรือเป็นเหตุให้บุคคลหรือร่างกายได้รับบาดเจ็บ Microsoft จะต้องรับผิดตามกฎหมายที่เป็นลายลักษณ์อักษร

ตามความในเงื่อนไขข้อที่ ii. Microsoft จะรับผิดเฉพาะในกรณีของการประมาทเลินเล่อที่ไม่ร้ายแรง ถ้า Microsoft ได้ทำการละเมิดพันธะสัญญาที่มีสาระสำคัญดังกล่าว การบรรลุเป้าหมายที่อำนวยความสะดวกในการปฏิบัติได้ครบถ้วนตามข้อตกลงฉบับนี้ การละเมิดที่อาจเป็นอันตรายต่อวัตถุประสงค์ของสัญญานี้และการปฏิบัติตามระเบียบข้อบังคับซึ่งคู่สัญญาฝ่ายหนึ่งอาจไว้วางใจได้เป็นประจำ (เรียกว่า "ภาระหน้าที่สำคัญ") ในกรณีของการประมาทเลินเล่อที่ไม่ร้ายแรงอื่น ๆ Microsoft จะไม่รับผิดจากการประมาทเลินเล่อเพียงเล็กน้อย

11. การปฏิเสธความรับผิดชอบในการรับประกัน ซอฟต์แวร์นี้ได้รับอนุญาตให้ใช้สิทธิ์ใน “แบบที่เป็น” คุณเป็นผู้รับความเสี่ยงในการใช้งาน เอง Microsoft ไม่ได้ให้การรับประกัน การรับรอง หรือเงื่อนไขที่แสดงนัยอย่างชัดแจ้ง Microsoft ละเว้นการรับประกันโดยนัยทั้งหมด ซึ่งรวมถึงความสามารถในเชิงพาณิชย์ ความสมบูรณ์เพื่อการใช้งานเฉพาะด้าน และการไม่ละเมิด ตามขอบเขตที่อนุญาตให้ใช้ตามกฎหมายที่บังคับใช้

12. ข้อจำกัดและข้อยกเว้นด้านความเสียหาย หากคุณมีมูลเหตุที่จะเรียกร้องค่าชดเชยความเสียหาย ถึงแม้จะมีถ้อยคำปฏิเสธความรับผิดชอบในการรับประกันดังที่กล่าวมาก่อนหน้านี้ก็ตาม คุณสามารถเรียกร้องค่าชดเชยจาก Microsoft และซัพพลายเออร์ของบริษัท ได้เฉพาะค่าความเสียหายโดยตรงสูงสุดไม่เกิน 5.00 ดอลลาร์สหรัฐ คุณไม่สามารถเรียกร้องความเสียหายอื่นใดที่เกิดขึ้น รวมถึงความเสียหายอันเป็นผลสืบเนื่อง การสูญเสียผลกำไร ความเสียหายพิเศษ ความเสียหายโดยอ้อม หรือความเสียหายอันเกิดจากการผิดสัญญา

ข้อจำกัดนี้มีผลใช้กับ (a) ทุกกรณีที่เกี่ยวข้องกับซอฟต์แวร์ บริการ เนื้อหา (รวมถึงรหัส) ที่เกิดบนอินเทอร์เน็ตไซต์ของบุคคลภายนอก (b) การเรียกร้องค่าเสียหายสำหรับการผิดสัญญา การรับประกัน การรับรอง หรือเงื่อนไข; ความรับผิดโดยสิ้นเชิง การประมาทเลินเล่อ หรือการละเมิดอื่น ๆ; หรือการเรียกร้องความเสียหายอื่น ๆ; ในแต่ละกรณีภายในขอบเขตที่กฎหมายที่บังคับใช้อนุญาต

ข้อจำกัดนี้ยังมีผลใช้แม้ว่า Microsoft ได้ทราบหรือควรทราบเกี่ยวกับความเป็นไปได้ที่จะเกิดความเสียหายดังกล่าว ข้อจำกัดหรือข้อยกเว้นข้างต้นอาจไม่นำมาใช้กับคุณได้เนื่องจากรัฐ จังหวัด หรือประเทศของคุณอาจไม่อนุญาตให้มีข้อยกเว้นหรือข้อจำกัดเกี่ยวกับความเสียหายอันเกิดจากการผิดสัญญา ความเสียหายอันเป็นผลสืบเนื่อง หรือความเสียหายรูปแบบอื่น
